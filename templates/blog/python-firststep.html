About Python Language.

Python is a general-purpose programming language with advanced features. Python was created in the late 1980 and its implementation was started in December 1989. Python 2.0 was released on October 16, 2000, and Python 3.0 was released on December 2008, with many major new features. Python is used in many applications, such as Web and internet development, Scientific and Numeric Application, Desktop Application and, in education domain also.
